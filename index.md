# Inżynieria Systemowa Linux & Architektura Kontenerowa

Projektuję, wdrażam i utrzymuję bezpieczną oraz skalowalną infrastrukturę serwerową w oparciu o systemy klasy enterprise. Specjalizuję się w automatyzacji, wirtualizacji, konteneryzacji, orchiestracji, usługach chmurowych i przepływie danych.

---

## 🛠 Zakres kompetencji

* **Administracja i Architektura systemów:** Wdrażanie, hardening i optymalizacja systemów operacyjnych z rodziny Linux, Windows oraz macOS.
* **Konteneryzacja i wirtualizacja Enterprise:** Budowa odizolowanych środowisk produkcyjnych w standardzie *rootless*, zintegrowanych natywnie z systemd za pomocą mechanizmów Quadlet. Tworzenie i zarządzanie maszynami wirtualnymi Proxmox, ESXi, vSphere, w tym także High Performance Computing (HPC).
* **Automatyzacja procesów:** Projektowanie i wdrażanie kompletnych pipeline'ów dla wszelakiej maści oprogramowania, testów automatycznych, wdrażania. Opracowanie procesów przepływu danych.
* **Rozwiązania chmurowe:** Konfiguracja bezpiecznych, wydajnych i zoptymalizowanych kosztowo rozwiązań opartych na chmurach AWS, Azure, GCP oraz OCI.

---

## 💻 Stack technologiczny

* **OS:** Linux, macOS, Windows
* **Kontenery i orchiestracja:** Podman, Quadlet, systemd, podman-compose, Rancher, 
* **Skryptowanie:** Bash, Python, R
* **Automatyzacja:** Jenkins, GitLab CI, GitHub Actions
* **Witualizacja:** Proxmox, ESXi, vSphere, HPC
